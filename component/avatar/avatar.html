Avatar

Avatars are images that users can set as their profile picture. On GitHub, they're always going to be rounded squares. They can be custom photos, uploaded by users, or generated as Identicons as a placeholder.

Example of Image Avater

Avatar is available in 4 different sizes.We can use image in Avatar. You need to include class avatar and for size add class according to size xl-avatar, large-avatar, medium-avatar, small-avatar (e.g. class="avatar xl-large")

Example of Text Avatar

We can use initial letters of user in Avatar as well. You need to include common class avatar and its size class and class pink for styling. (e.g. class="avatar text-xl-avatar pink")
